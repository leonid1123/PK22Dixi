КОНТРОЛЬНАЯ РАБОТА
Разработать программу на языке C# по следующим требованиям:
1. консольное приложение
2. Программа должна содержать массив элементов типа Int
3. Программа должна содержать массив элементов типа String
4. Элементы массива типа String пользователь задает 
при помощи консольного диалога.
5.  Элементы массива типа Int пользователь задает по формуле:
    x[i]=2*i*i-12*Int
6. Программа выводит в консоль строку типа:
"Элемент" {значение элемента массива Int} "содержит значение" {значение элемента массива String}
